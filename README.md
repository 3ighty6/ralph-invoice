# Ralph Invoice — Ralph & Sons LLC

Invoicing, client management, and PDF export. Built by Ralph & Sons LLC.

## Deploy to Netlify (One Time Setup)

1. Push this folder to a GitHub repo named `ralph-invoice`
2. Go to netlify.com → Add new site → Import from Git
3. Select the repo → Build command: `npm run build` → Publish dir: `build`
4. Click Deploy

That's it. Live URL in 2 minutes.
